CSD-430 Module 2 — JSP Scriptlets Assignment (VS Code/Tomcat)
================================================================

Place these files in your local assignment folder:
C:\Users\K1tt3\csd\CSD-430\Module-2

Structure:
  country.jsp
  assets\styles.css
  CrystalLong_Module2_Screenshots.docx (you will create this)

Run (VS Code only):
1) Install "Tomcat for Java" and "Extension Pack for Java" extensions.
2) Ctrl+Shift+P → Tomcat: Add Tomcat Server → select your Tomcat folder.
3) In the Tomcat panel, Start the server.
4) Right-click the server → Add Webapp → choose your Module-2 folder → context path: /module-2
5) Open http://localhost:8080/module-2/country.jsp

Zip for submission:
  Select:
    country.jsp
    assets\styles.css
    CrystalLong_Module2_Screenshots.docx
  Right-click → Send to → Compressed (zipped) folder → rename to:
    long_mod_2_csd430.zip
